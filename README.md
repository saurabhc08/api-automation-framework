# API Automation Framework

This project is built using Rest Assured and TestNG.

## Features
- API Automation (GET, POST)
- Token-based Authentication
- API Chaining
- Config-driven framework

## Tech Stack
- Java
- Rest Assured
- TestNG
- Maven

## Project Structure
src/test/java
- api.base
- api.tests
- api.utils
- api.constants

src/test/resources
- config.properties
